# Brand & Marketing Skills

Skills for brand identity, marketing content, and social media.

## Skills

| Skill | Description | Auto-Invoke |
|-------|-------------|-------------|
| `card-twitter` | Twitter quote/data card designer | "create a twitter card" |
| `card-xiaohongshu` | Xiaohongshu knowledge cards | "create a xiaohongshu card" |
| `competitive-ads-extractor` | Extract competitor ads | "analyze competitor ads" |
| `domain-name-brainstormer` | Generate domain names | "brainstorm domain names" |
| `marketing-psychology` | Apply psychology to copy | "improve marketing copy" |
| `raffle-winner-picker` | Random winner selection | "pick a raffle winner" |
| `slack-gif-creator` | Animated GIF for Slack | "create a slack gif" |
| `social-reddit-card` | Reddit post card design | "create a reddit card" |
| `social-spotify-card` | Spotify now playing card | "create a spotify card" |
| `social-x-post-card` | X post card design | "create an x post card" |
| `tailored-resume-generator` | Custom resume creation | "generate a resume" |
| `twitter-algorithm-optimizer` | Optimize for Twitter algo | "optimize twitter content" |

## Installation

Skills are installed to `~/.agents/skills/`. Each skill is self-contained with its own `SKILL.md`.

## Usage

Skills auto-invoke when you describe the task. Example:
- "Create a Twitter card for my new blog post"
- "Analyze competitor ads on Facebook"
- "Generate a resume for a senior engineer role"
