# Code Quality Skills

Skills for code review, debugging, testing, and quality assurance.

## Skills

| Skill | Description | Auto-Invoke |
|-------|-------------|-------------|
| `audit-project` | Comprehensive project audit | "audit this project" |
| `autofix` | Auto-fix code issues | "autofix these issues" |
| `bug-debugging` | Debug reproducible bugs | "debug this bug" |
| `bug-fix` | Quick bug fixes | "fix this bug" |
| `code-review-and-quality` | Multi-axis code review | "review this code" |
| `code-review` | Quick code review | "review my changes" |
| `code-simplification` | Simplify complex code | "simplify this code" |
| `debugger` | Debugging specialist | "help me debug" |
| `debugging-and-error-recovery` | Systematic debugging | "debug this error" |
| `doubt-driven-development` | Adversarial review | "challenge this approach" |
| `hallucination-debugging` | Debug AI hallucinations | "debug hallucination" |
| `pr-feedback-quality-gate` | PR feedback quality | "review PR feedback" |
| `pr-review` | Pull request review | "review this PR" |
| `rag-evaluation` | Evaluate RAG systems | "evaluate RAG quality" |
| `systematic-debugging` | Structured debugging | "systematically debug" |
| `test-driven-development` | TDD workflow | "write tests first" |
| `test-generation` | Generate test cases | "generate tests" |
| `test-writer-fixer` | Write/fix tests | "write tests for this" |
| `verification-before-completion` | Verify before claiming done | "verify this works" |

## Installation

Skills are installed to `~/.agents/skills/`. Each skill is self-contained with its own `SKILL.md`.

## Usage

Skills auto-invoke when you describe the task. Example:
- "Review my code for security issues"
- "Debug this failing test"
- "Simplify this complex function"
